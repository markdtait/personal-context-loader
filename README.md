# Personal Context Loader

Automatically loads your personal context document from Google Drive at the start of every Cowork session. Claude always knows who you are, how you communicate, and what you're working on — with zero effort.

## Install
/plugin marketplace add markdtait/personal-context-loader
/plugin install personal-context-loader@personal-context-loader

## Setup
Connect Google Drive, then say "set up my context" to create your context doc.
